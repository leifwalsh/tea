Wraps an underlying std::io::Write so that bytes written get encrypted and passed through. You must call close() when finished writing to append the padding bytes.

Example:

use std::fs::File;
use std::io::Write;
use tea::io::Writer;

let f = File::create("foo.txt").ok().unwrap();
let mut crypt_f = Writer::new(f, [1, 2, 3, 4], [5, 6]);
crypt_f.write_all(b"Hello, world!").ok().unwrap();
crypt_f.close().ok().unwrap();

Trait Implementations

impl<W: Write> Write for Writer<W>

fn write(&mut self, buf: &[u8]) -> Result<usize>

Encrypts the bytes in buf and passes them through to the underlying std::io::Write. If there are not an exact multiple of 8 bytes available, the remaining ones will be cached until more data is written or the Writer is closed.

fn flush(&mut self) -> Result<()>

Passes the flush call through to the underlying std::io::Write object, but only if the internal buffer is clear. If we have some cached bytes that are waiting for a full block before they can be encrypted, it is an error to try to call flush().
